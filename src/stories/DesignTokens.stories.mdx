# Design Tokens

Design tokens são a menor parte de um Design System, garantindo a mesma comunicação visual, de maneira íntegra e distribuída entre todos os componentes que fazem parte de um sistema.

> A Design System is the single source of truth which groups all the elements that will allow the teams to design, realize and develop a product. [`HACQ, Audrey (UX Collective, 2018)`](https://uxdesign.cc/everything-you-need-to-know-about-design-systems-54b109851969)

#### Font Size

A hierarquia tipográfica do DataLab foi definida a partir da proporção de 1.250 - também conhecida como Major Third - a partir de um tamanho base de 16px (16px = 1em). Construída dessa forma, a escala tipográfica dispõe de contraste e harmonia entre cada intervalo.
