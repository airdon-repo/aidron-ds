# Radio
Radios são utilizados quando, dentro de uma variedade de opções, apenas uma pode ser selecionada. Se há apenas uma opção para ser selecionada, usa-se o checkbox. É um componente exclusivo para Web e Android.
<Canvas>

  >
    <RadioOption value="1">Option 1</RadioOption>

    <RadioOption value="3">Option 3</RadioOption>
  </Radio>
</Canvas>

## Anatomia
* Check icon ou Indeterminate icon (apenas quando selecionado).
* Checkbox.
* Label.
* Hit area.
